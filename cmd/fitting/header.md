Here is a list of header terms in the fitting result file.

* `group`: name of the gene set
    - `all`: all genes
    - `boot_X`: bootstrap of random sampling of gene subset
* `sample_d`: $d\sb{sample}$
* `theta`: $\theta\sb{pool}$
* `phi`: $\phi\sb{pool}$
* `fbar`: $\bar f$
* `ratio`: $\gamma/\mu$
* `rho`: $\gamma \bar f$
* `sample_theta`: $\theta\sb{sample}$
* `sample_rho`: $c$
